
# React Native OTP Input (Expo Version)

![Tests](https://github.com/Twotalltotems/react-native-otp-input/workflows/Tests/badge.svg)

 This library was forked from **@twotalltotems/react-native-otp-input** and modified to support expo built apps. 

![demo.gif](https://s3.ca-central-1.amazonaws.com/tttevents/iosvideo.gif)
![demo.gif](https://s3.ca-central-1.amazonaws.com/tttevents/android.gif)

## Installation

`npm install --save https://github.com/brvhprince/react-native-otp-input.git`
or
`yarn add https://github.com/brvhprince/react-native-otp-input.git`

## Dependencies

`npm install --save expo-clipboard`
or
`yarn add expo-clipboard`

## Basic Usage

```js
import OTPInputView from '@brvhprince/react-native-otp-input'

...

<OTPInputView pinCount={4} />

```
